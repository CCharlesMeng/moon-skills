# 后端检查清单

适用信号：需求涉及 API、数据、权限、可靠性。

## 检查项

- [ ] 契约是否变化，兼容边界在哪里
- [ ] 幂等、重试、超时和失败恢复是否明确
- [ ] 数据一致性和权限校验是否受影响
- [ ] 是否需要迁移、回填、双写或灰度
- [ ] 可观测性是否足以支持上线后定位

## 定制

要修改本检查清单，复制本文件到 `.project-context/lenses/backend.md` 并编辑。同名文件仓库版本优先于 skill 默认。不要直接修改本文件。

要新建检查清单，参考 [lens-template.md](lens-template.md)。
